## Backup / If Something Breaks

- **Pipeline crashes:** The pre-baked pickle loads in < 1 second. Check `models/audit_result_demo.pkl` exists.
- **Gemini narrative fails:** Fallback rule-based narrative auto-generates. Just click the button.
- **Port conflict:** `streamlit run ui/app.py --server.port 8502`
- **ImportError:** Activate venv: `venv\Scripts\activate` then `pip install -r requirements.txt`
